# Run Generated Code

This guide explains how to run the code generated by API Dash for different programming languages.

## Python

Installation: [https://www.python.org/downloads/](https://www.python.org/downloads/)

Install dependency (`requests`):

```bash
pip install requests
```

Run generated file:

```bash
python generated_request.py
```

## JavaScript (Node.js)

Installation: [https://nodejs.org/en/download](https://nodejs.org/en/download)

Install dependency (`axios`):

```bash
npm install axios
```

Run generated file:

```bash
node generated_request.js
```

## Java

Installation: [https://adoptium.net/temurin/releases/](https://adoptium.net/temurin/releases/)

Install dependency (`okhttp`):

```bash
mvn dependency:get -Dartifact=com.squareup.okhttp3:okhttp:4.12.0
```

Run generated file:

```bash
javac GeneratedRequest.java
java GeneratedRequest
```
